# Red Nose Reports

This repository contains a solution for **[Advent of Code 2024 – Day 2](https://adventofcode.com/2024/day/2)**.

## Files
- **`solution1.js`** – Main solution script for part 1
- **`solution2.js`** – Main solution script for part 2 
- **`input.txt`** – Input data file ([download your personal input here](https://adventofcode.com/2024/day/2/input))

## How to Run
1. Navigate to this project directory in your terminal.  
2. Run the script using Node.js:

   ```bash
   node solution1.js
   node solution2.js
   ```
